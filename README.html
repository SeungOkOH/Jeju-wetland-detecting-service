IBM ILOG CPLEX Optimization Studio 22.1.1

IBM® ILOG® CPLEX® Optimization Studio 는 계획 및 스케줄링 문제점의 전체 범위에 대해 효율적인 최적화 모델 및 최신 애플리케이션을 빌드하는 가장 빠른 방법을 제공합니다. 통합 개발 환경, 설명 모델링 언어, 내장 도구 및 수학적 프로그래밍 및 제한조건 프로그래밍 모두에 대한 최신 솔루션을 사용하여 전체 모델 개발 및 솔루션 프로세스를 지원합니다.

CPLEX 최적화 스튜디오는 여러 컴포넌트들을 포함한다. 각 구성요소는 설치 위치의 서브폴더 ( Linux의 서브디렉토리) 에 있습니다. 각 구성요소의 API및 커넥터는 구성요소 서브폴더 (서브디렉토리) 에 있습니다.

  • opl: OPL 모델링 환경 및 IDE (Integrated Development Environment)
  • cplex: 심플렉스, 배리어 및 혼합 정수 최적화 프로그램을 포함하는 CPLEX Optimizer
  • cpoptimizer: 제한조건 프로그래밍 기술을 사용하는 CP 최적화 프로그램
  • concert: 최적화 프로그램에서 사용할 C++ 모델링 계층
  • python: CPLEX및 CP 최적화 프로그램용 Python 모델링 계층 DOcplex 설치

기타 디렉토리:

  • doc: 문서의 HTML 버전
  • license: 라이센스 계약 문서
  • swidtag: 자원 명세 정보
  • Uninstall: 제품 및 설치 로그를 설치 제거하는 데 필요한 정보

Windows에 설치하기

Windows의 경우, IBM ILOG CPLEX Optimization Studio 의 기본 설치 위치는 다음과 같습니다.

C:\Program Files\IBM\ILOG\CPLEX_Studio2211

Community Edition 의 설치 위치는 다음과 같습니다.

C:\Program Files\IBM\ILOG\CPLEX_Studio_Community2211

설치 중에 다른 위치를 선택했을 수 있습니다. 이 위치는 이 문서의 나머지 부분에서 [install_dir] 로 참조됩니다.

중요 정보

CPLEX 최적화 스튜디오는 Windows 플랫폼에서 단일 포트로 제공됩니다. 해당 포트 ( x64_windows_msvc14) 는 Visual Studio 2019및 Visual Studio 2022와 호환 가능합니다. Visual Studio 버전과 서로의 호환성에 대한 자세한 내용은 Visual Studio 버전 간 C++ 바이너리 호환성을 참조하십시오.

이 제품에는 Microsoft Visual C++ Redistributable 패키지 (x64 버전) 가 필요합니다. Microsoft Visual C++ Redistributable Package (x64)패키지를 다운로드할 수 있습니다.

IDE

Linux

Linux 설치 프로그램은 실행 파일이며 설치를 실행하려면 파일에 실행 권한이 있어야 하며 명령행에서 실행해야 합니다. 예를 들어, ./<installername>.bin명령을 실행하십시오. 여기서 <installername> 은 설치 프로그램의 이름입니다. chmod u+x <installername>.bin명령을 사용하여 파일 실행 권한을 제공할 수 있습니다.

Linux IBM ILOG CPLEX Optimization Studio 의 기본 설치 위치는 다음과 같습니다.

/opt/ibm/ILOG/CPLEX_Studio2211

Community Edition 의 설치 위치는 다음과 같습니다.

/opt/ibm/ILOG/CPLEX_Studio_Community2211

IDE

Linux IDE를 실행할 수 있습니다.

1. 터미널 창 열기

2. 디렉토리를 [install_dir]/opl/oplide 로 변경하십시오.

3. 명령을 입력하십시오. ./oplide

또는 터미널 위치에서 절대 경로를 입력하십시오.

[install_dir]/opl/oplide/oplide
